Проект автоматизации Сервисного центра «Ремонт бытовой техники»

Выполнил: [студент группы К520с11-2 Чанкселиани Роланд]

О проекте
Разработка прототипа веб-ресурса для сервисного центра «Ремонт бытовой техники», специализирующегося на ремонте и обслуживании бытовой электроники. Цель проекта — автоматизировать процесс приема заявок на ремонт, отслеживания статуса заказов, управления складскими запасами запчастей и взаимодействия с клиентами. Это позволит сократить время обработки запросов, повысить эффективность работы и улучшить качество обслуживания.

Структура репозитория
/Analytics — Модели бизнес-процессов (AS IS / TO BE) в нотации BPMN 2.0 для сервисного центра.
/Prototypes — Вайрфреймы ключевых экранов (клиентский портал, панель администратора, форма заявки) и иерархическая структура (Сайтмэп).
/Docs — Сценарии использования, техническая документация и отчет о юзабилити-тестировании (Этап 3).
